python基础知识

  • dict.get(key, default=None) # key键值，default-若键值不存在，则返回该值
    返回指定键的值，如果值不在字典中返回默认值None。

  • str.join(sequence) # str- 通过该符号连接，sequence -- 要连接的元素序列。
    返回通过指定字符连接序列中元素后生成的新字符串。

  • os.listdir(path) # 以列表形式返回路径下文件

  • dataset = tf.data.Dataset.from_tensor_slices((data, label))

  • dataset = dataset.shuffle(buffer_size=10).batch(batch_size)

  • dataset = dataset.repeat()  # 重复
 it = dataset.__iter__() # 产生迭代器也可用data = next(iter(dataset))
 for i in range(20):
     x, y = it.next()
     print(x, y)

  • np.max() 与np.min()

    当对象是numpy.ndarray时，最值函数无参数时是求整体的最大最小值，参数axis = 0,求每列的最值，axis = 1时，求每一行的最值。而对于DataFrame格式时，默认无参数时则是求每一列的最值。
