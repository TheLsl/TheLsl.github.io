1.常用tensorflow的损失函数

常用损失函数有类实现和函数实现两种，类实现一般单词首字母大写，函数实现为小写。调用类损失函数需要实例化。CategoricalCrossentropy 和 categorical_crossentropy 都是类别交叉熵损失函数，前者是类的实现形式，后者是函数的实现形式。一般包含以下：
官方链接
类实现的损失函数，输出结果是自动对batch取均值，函数实现的损失函数输出为每个样本的误差，而非一个batch的平均误差

  • mean_squared_error（平方差误差损失，用于回归，简写为 mse, 类与函数实现形式分别为为 MeanSquaredError 和 MSE）
    loss = mean(square(y_true - y_pred), axis=-1)

  • mean_absolute_error (绝对值误差损失,用于回归,简写为 mae, 类与函数实现形式分别为 MeanAbsoluteError 和 MAE)

  • mean_absolute_percentage_error (平均百分比误差损失，用于回归，简写为 mape, 类与函数实现形式分别为 MeanAbsolutePercentageError 和 MAPE)

  • Huber(Huber损失，只有类实现形式，用于回归，介于mse和mae之间，对异常值比较鲁棒，相对mse有一定的优势)

  • binary_crossentropy(二元交叉熵，用于二分类，类实现形式为 BinaryCrossentropy)

  • categorical_crossentropy(类别交叉熵，用于多分类，要求label为onehot编码，类实现形式为 CategoricalCrossentropy)

  • sparse_categorical_crossentropy(稀疏类别交叉熵，用于多分类，要求label为序号编码形式，类实现形式为 SparseCategoricalCrossentropy)

  • hinge(合页损失函数，用于二分类，最著名的应用是作为支持向量机SVM的损失函数，类实现形式为 Hinge)

  • kld(相对熵损失，也叫KL散度，常用于最大期望算法EM的损失函数，两个概率分布差异的一种信息度量。类实现形式为 KLDivergence 或 KLD)

  • cosine_similarity(余弦相似度，可用于多分类，类实现形式为 CosineSimilarity)

2. tf.nn.实现的交叉熵误差

  • tf.nn.sigmoid_cross_entropy_with_logits(_sentinel=None,labels=None, logits=None, name=None)
  1. 参数说明：logits为网络的输出，且结果未通过sigmoid激活函数，shape为[batch,num_class]
     labels为实际的样本标签，shape为[batch,num_class]
  2. labels 和 logits两者必须有一样的shape，且数据类型同为[float16,float32,float64]
  • tf.nn.softmax_cross_entropy_with_logits(_sentinel=None, labels=None, logits=None, dim=-1, name=None)
  1. 该函数的labels需为one-hot型，且每个样本只有一个1
  2. logits同样为未通过激活函数的输出值，且形状，数据类型与labels一致
  • tf.nn.sparse_softmax_cross_entropy_with_logits(_sentinel=None,labels=None,logits=None, name=None)
  1. 该函数的labels不用one-hot型，函数会自动将标签转换为one-hot型
  2. logits同样为未通过激活函数的输出值，且形状，数据类型与labels一致
